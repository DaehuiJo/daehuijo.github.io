Hexo와 Github Page를 이용한 블로그 개설


[이미지 출처: http://masatolan.com/images/2015/programming/1/hexo-blog-basic1.jpg]

블로그 개설을 결심하고 Tistory를 기웃거리면서 구글링 중 github page를 이용한 블로그에 대해 알게 되었다.

Jekyll(Ruby 기반), Hugo(Go 기반), Hexo(Node.js 기반)와 같은 Static Website Generator를 이용하여 생성한 블로그를 github에 push하면 블로그 사이트가 만들어 진다.

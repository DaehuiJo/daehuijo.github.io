Spring 개발환경 설정

2017-04-03
framework, spring

[이미지 출처: https://spring.io/]

실습환경

  • OS: Ubuntu 16.04.2 LTS
  • JDK: 1.8.0_121
  • IDE: Eclipse_Neon.3 Release (4.6.3)

Spring IDE 설치

1) Eclipse [Menu] -> [Help] -> [Install New Software]

2) [Work with]란에http://dist.springsource.com/release/TOOLS/update/e4.6/ 입력

3) 설치할 software를 선택 후 [Next]

4) License 내용 [accept] 선택 후 [Finish]

STS Plugin 설치

1) Eclipse [Menu] -> [Marketplace]

2) STS 검색 후 Install

또는, 공식 웹사이트 marketplace에서 install tab을 Eclipse로 Drag 하여 설치
